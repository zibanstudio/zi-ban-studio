# Zi Ban Portfolio Website

This project is a fully responsive portfolio website built using HTML, CSS, and JavaScript.

The goal of this project was to create a clean and structured platform to present visual work, while focusing on layout, typography, and user experience.

---

## Overview

The website includes five main pages:

- Home  
- About  
- Work  
- Media  
- Contact  

Each page is designed with a consistent navigation system and visual hierarchy, allowing smooth transitions and clear readability.

---

## Features

- Responsive layout using Flexbox and CSS Grid  
- Custom navigation and hover interactions  
- Embedded media using iframe (external content)  
- Functional contact form using FormSubmit  
- Google Analytics 4 integration  
- Custom typography using Google Fonts  
- Scroll-based reveal animations  
- Structured layout for project-based content  

---

## What I’m Proud Of

I focused on building a layout that feels intentional rather than template-based.

The project highlights how visual work can be presented through structure and pacing, not just images.  
I paid particular attention to spacing, typography, and how each section flows across the site.

---

## Challenges

One of the main challenges was balancing design and code.

Translating visual ideas into CSS layout (especially spacing and responsiveness) required multiple iterations.  
I also worked through issues related to asset loading, layout consistency, and cross-page styling.
